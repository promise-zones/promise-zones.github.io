    Welcome to Promise Zones on Github

    This started as a 2016 National Day of Civic Hacking Challenge: Data Visualization for Obama Administration's Promise Zones.

    Main Visualization

    The main (and currently only) visualization we have made so far, compares the unemployment rate to the poverty rate, for each county in the US, and compares those to the Promise Zones. You can see that here (todo: add link when it's set up).

    Future Plans

    See the Github Issues for a list of things we plan to do. In the grand scheme of things, we plan to iterate and improve the main visualization, and hope to link it to something based on a map. We may add additional visualizations in the future based on different data points (and we are open to suggestions!).

    Authors and Contributors

    The project started with @dksmith01, @jonyo, @nkcmr, and @stuagano. We made the decision to push the project forward, continuing to improve and add to the visualization we came up with.

    We are open to additional contributors if you are interested in helping our project out!

    To be clear, we are doing this on our spare time (similar to people donating their time to help any time of civic endeavor like volunteer at soup kitchen, etc). To be even more clear, this is not directly run by the Obama Administration or the government.
